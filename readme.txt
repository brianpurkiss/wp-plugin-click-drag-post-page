== Description ==

Click and Drag to re-order posts and pages from within the WordPress Admin, reordering them within The Loop. Useful for people who want to easily re-order content.

== Installation ==

1. Upload the plugin zip to the `/wp-content/plugins/` directory, or upload through the WordPress Admin's plugin page.
1. Activate the plugin through the 'Plugins' menu in WordPress
